# agency-api
This is a backend-only application designed to manage and handle relationships among Assets, Users, and Agencies.

Stack:
- Node.js +18.x
- Express
- MongoDB
- Jest/Supertest

### Usage:
1. npm i
2. create .env
3. For testing:
    - npm test
4. For developing:
    - npm run dev
5. For prod:
    - npm start
